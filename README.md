# Cyber Shield X

## Overview

Cyber Shield X is a web-based cybersecurity monitoring platform developed to analyze logs, inspect network traffic, perform reconnaissance, and assess website security. The platform helps security professionals identify suspicious activities and investigate potential threats through an easy-to-use dashboard.

## Features

### Log Analysis

* Upload and analyze system log files
* Detect suspicious IP addresses
* Identify brute-force login attempts
* Highlight unauthorized access attempts

### Network Reconnaissance

* Scan target hosts for open ports
* Discover active network services
* Gather network intelligence

### Packet Inspection

* Analyze PCAP files
* Inspect packet payloads using Scapy
* Detect suspicious traffic patterns

### Website Security Analysis

* Extract links, images, headings, and forms
* Collect information useful for security assessment

### Reporting

* Export analysis results in JSON format
* Generate structured reports for investigation
